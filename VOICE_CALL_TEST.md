### 📝 技术细节

**音频处理流程:**
1. 麦克风 → getUserMedia API
2. AudioWorklet → 实时音频处理
3. WebCodecs Encoder → Opus编码
4. WebSocket → 发送到服务器
5. 服务器 → 转发给其他客户端
6. WebSocket → 接收音频数据
7. WebCodecs Decoder → Opus解码
8. AudioContext → 播放音频

**性能优化:**
- 使用AudioWorklet避免主线程阻塞
- WebCodecs硬件加速编解码
- Opus高效压缩（32kbps）
- WebSocket二进制传输
